React

Bibliothèque UI Frontend

React est une librairie javascript open source aidant au développement d’interface web monopages. Cette librairie est développée et maintenue par Meta et la communauté React depuis 2013.

Comme Angular, cette librairie permet de découper ses interfaces en composants que l’on peut agréger et réutiliser. React s’appuie aussi sur le modèle MVC mais ne peut pas gérer tout le MVC tout seul, il faudrait par exemple le coupler redux pour gérer des données par exemple.

Bien que je sois moins familier avec React qu’avec Angular, j’ai tout de même eu l’occasion de pratiquer cette librairie dans un projet en entreprise et un projet d’école qui nous avait permis d’obtenir un financement pour une association.

Anecdotes :

Contrairement à Angular, React est une librairie avec laquelle j’ai moins eu l’occasion de travailler. Le projet illustrant le mieux mes capacités en react est le projet Spinnin’ dans lequel j’ai réalisé un site vitrine pour la saladerie Bodeli.

Ce projet m’a permis de découvrir les principes de react en tant que librairie front. J’ai donc travaillé sur la réalisations de composants indépendants qui ont été par la suite mis ensemble pour constituer ce site vitrine. Le développement de la navigation et d’un carrousel de photos m’a permis d’utiliser des fonctionnalités de react comme les points d’ancrage dans le DOM.

Autocritique :

React est une compétence que j’ai moins développée que les autres, surtout quand on la compare à mon autre compétence de développement frontend qu’est Angular. J’ai appris des bases suffisantes sur ce framework pour pouvoir continuer d’apprendre dessus en autonomie et pouvoir me former dessus si mes expériences professionnelles futures le demandent. C’est donc pour cette raison que je considère avoir acquis un niveau autonome sur React.

React est pour moi une compétence que je devais maîtriser pour entrer sur le marché du travail, en raison de son utilisation très fréquente dans l’environnement moderne du développement logiciel. Et je compte approfondir mes compétences grâce à des projets personnels une fois mes études terminées.
