React

Bibliothèque UI Frontend

React est une librairie javascript open source aidant au développement d’interface web monopages. Cette librairie est développée et maintenue par Meta et la communauté React depuis 2013.

Comme Angular, cette librairie permet de découper ses interfaces en composants que l’on peut agréger et réutiliser. React s’appuie aussi sur le modèle MVC mais ne peut pas gérer tout le MVC tout seul, il faudrait par exemple le coupler redux pour gérer des données par exemple.

Bien que je sois moins familier avec React qu’avec Angular, j’ai tout de même eu l’occasion de pratiquer cette librairie dans un projet en entreprise et un projet d’école qui nous avait permis d’obtenir un financement pour une association.
